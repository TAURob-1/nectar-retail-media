# "Build Your Own Agent" Platform Comparison

## Summary

Comparison of platforms offering "build your own AI agent" capabilities, including marketing-focused platforms, general-purpose agent builders, and big-tech studio offerings.

---

## 1. NinjaCat
**Focus:** Marketing & Advertising Agents
**URL:** https://www.ninjacat.io/marketing-performance-management-platform/ai-agents
**Screenshots:** `screenshots/ninjacat-agents.png`, `screenshots/ninjacat-agent-builder.png`

| Feature | Details |
|---------|---------|
| Agent Builder | Visual, no-code. Two paths: recruit from gallery of pre-built templates, or build custom from scratch |
| Target User | Marketing agencies, brands |
| Data Sources | Unified Data Cloud + connected marketing sources |
| Agent Types | PPC hygiene, creative checks, spend monitoring, taxonomy audits, benchmarking, client comms |
| Time to Deploy | Under 1 hour for first agent |
| White Label | Enterprise platform, not explicitly white-label |
| Key Differentiator | Marketing-specific — agents trained on your unified ad/analytics data |

---

## 2. Akkio
**Focus:** White-Label Generative BI & Analytics for Agencies
**URL:** https://www.akkio.com/agency
**Screenshot:** `screenshots/akkio-platform.png`

| Feature | Details |
|---------|---------|
| Agent Builder | No-code, chat-based data exploration. Build custom ML models via UI |
| Target User | Digital agencies, consultants |
| Data Sources | Connect client data, create branded dashboards |
| White Label | Yes — custom URL or fully embeddable API (Build-On Package) |
| Key Features | Chat Explore (natural language data queries), predictions, forecasts, branded reports |
| Key Differentiator | White-label first — agencies embed Akkio into their own product/service under their brand |

---

## 3. MindStudio
**Focus:** General-Purpose No-Code AI Agent Builder
**URL:** https://www.mindstudio.ai/
**Screenshot:** `screenshots/mindstudio.png`

| Feature | Details |
|---------|---------|
| Agent Builder | Visual workflow builder, no-code. Describe agent and "Agent Architect" scaffolds it |
| Target User | Anyone — business users to developers |
| AI Models | 200+ models (OpenAI, Claude, Gemini, etc.) — no separate API keys needed |
| Deployment | Web apps, backend automations, browser extension, email, API endpoints (Zapier/Make/n8n) |
| Templates | 100+ pre-built templates |
| Time to Build | 15 minutes to 1 hour |
| Compliance | SOC 2, GDPR |
| Key Differentiator | Model-agnostic, Dynamic Tool Use — agents autonomously select and combine tools |

---

## 4. Relevance AI
**Focus:** Multi-Agent Workflows for Sales & GTM
**URL:** https://relevanceai.com/agents
**Screenshot:** `screenshots/relevance-ai.png`

| Feature | Details |
|---------|---------|
| Agent Builder | Low-code "chains" linking actions into workflows |
| Target User | Sales/GTM teams, agencies |
| Multi-Agent | Agents form teams — one finds info, another verifies, third writes report |
| Templates | BDR Agent, Account Research Agent, etc. |
| Pricing | Free tier (200 actions/mo), Pro/Team/Business up to $349+/mo. Credit-based |
| Key Differentiator | Multi-agent orchestration — agents collaborate on complex tasks |

## 5. Lyzr
**Focus:** Enterprise AI Agent Studio
**URL:** https://www.lyzr.ai/lyzr-agent-studio/
**Screenshot:** `screenshots/lyzr-agent-studio.png`

| Feature | Details |
|---------|---------|
| Agent Builder | No-code studio with ready-made blueprints |
| Target User | Enterprise (banking, logistics, SaaS) |
| AI Models | Model-agnostic — choose per use case |
| Security | SOC 2, GDPR, HIPAA, ISO 27001. 100% data privacy with flexible deployment |
| Agent Types | Solo agents, multi-agent systems, full orchestration |
| Blueprints | Sales, support, ops, finance workflows |
| Key Differentiator | Enterprise compliance-first — Safe AI and Responsible AI baked into core architecture |

## 6. Google Vertex AI Agent Builder
**Focus:** Enterprise-Grade Agent Development Platform
**URL:** https://cloud.google.com/products/agent-builder
**Screenshot:** `screenshots/google-vertex-agent-builder.png`

| Feature | Details |
|---------|---------|
| Agent Builder | Agent Designer (low-code visual), Agent Development Kit (ADK — code-first, open-source) |
| Target User | Developers, enterprises |
| AI Models | Gemini models |
| Deployment | Agent Engine — fully managed runtime, evaluation, sessions, memory bank, code execution |
| Features | Agent Garden (library of sample agents), multi-agent workflows, Python/Java/more |
| Scale | Global scale, production-grade |
| Key Differentiator | Full Google Cloud ecosystem — ADK is open-source, production deployment built-in |

## 7. Google Workspace Studio
**Focus:** No-Code Agent Building for Workspace Users
**URL:** https://workspace.google.com/studio/
**Screenshot:** `screenshots/google-workspace-studio.png`

| Feature | Details |
|---------|---------|

| Target User | Business users (all Workspace Business & Enterprise plans) |
| Integration | Native Google Workspace (Gmail, Drive, Docs, Sheets, Calendar) |
| Time to Build | Minutes |
| Availability | GA with all Business and Enterprise Workspace plans |

## 9. Claude (Anthropic) — Cowork / Agent SDK
**Focus:** AI Agent Development via API & Cowork Platform
**URL:** https://claude.com/solutions/agents
**Screenshot:** `screenshots/claude-cowork.png`

| Feature | Details |
|---------|---------|
| Agent Builder | Claude Cowork (agentic AI workflows), Agent SDK (code-first), Claude Code subagents |
| Target User | Developers, teams using Claude |
| AI Models | Claude Opus, Sonnet, Haiku |
| Multi-Agent | Agent Teams — orchestrator spawns up to 10 sub-agents working in parallel |
| Features | File editing, code execution, tool calling, MCP integration, Skills marketplace |
